# 💕 恋爱小窝 - 李安 ❤️ 韩舒薇

一个充满爱意的互动网站，一起养一只虚拟小宠物！

## ✨ 功能

- 🐾 **虚拟宠物养成** - 喂食、玩耍、哄睡、亲亲，一起照顾小宠物
- 🔄 **实时同步** - 你喂食，她立刻看到（支持多设备同步）
- 📈 **宠物进化** - 根据照顾程度，宠物有5个进化阶段
- 💌 **恋爱小纸条** - 互相留言，写下甜蜜的话语
- 📋 **互动记录** - 查看谁做了什么，温馨的互动日志
- 💝 **纪念日倒数** - 记录在一起的天数
- 🎨 **精美动画** - 漂浮爱心、宠物表情变化、进化特效
- ⌨️ **快捷键** - F=喂食 P=玩耍 S=哄睡 L=亲亲

## 🚀 快速开始

### 方式一：直接使用（无需后端）

直接打开 `index.html` 即可使用！数据保存在浏览器中。

### 方式二：部署到 GitHub Pages（推荐）

1. 在 GitHub 创建新仓库（如 `love-nest`）
2. 推送代码：
   ```bash
   git init
   git add .
   git commit -m "💕 我们的恋爱小窝"
   git remote add origin https://github.com/你的用户名/love-nest.git
   git push -u origin main
   ```
3. 在仓库 Settings → Pages 中启用 GitHub Pages
4. 选择 `main` 分支，点击 Save
5. 等待几分钟，访问 `https://你的用户名.github.io/love-nest/`

### 方式三：启用实时同步（Supabase）

1. 在 [Supabase](https://supabase.com) 创建免费账号和项目
2. 在 SQL Editor 中运行 `supabase-schema.sql`
3. 在 Settings → API 中复制 URL 和 anon key
4. 编辑 `js/app.js`，替换 `SUPABASE_CONFIG` 中的值
5. 重新部署

## 📱 使用说明

1. 打开网站后，点击"李安"或"韩舒薇"选择身份
2. 点击按钮照顾宠物（或使用快捷键）
3. 宠物属性会随时间缓慢下降，记得常来看看
4. 给小宠物起个名字：点击右下角⚙️设置
5. 设置纪念日：在设置中修改
6. 分享链接给另一半：加上 `#han` 自动选韩舒薇，`#li` 选李安
